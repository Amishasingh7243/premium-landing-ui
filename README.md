# Premium Landing Page — Next.js + Tailwind (Upgraded UI)

This project is a premium, responsive landing page starter built with **Next.js (pages router)**, **Tailwind CSS**, and **Framer Motion** for animations.

## Run locally
1. `npm install`
2. `npm run dev` (http://localhost:3000)

## Build
`npm run build` then `npm run start` for production.

## Notes
- Animations use `framer-motion` and components include `use client` where required.
- Designed for quick deployment to Vercel (connect repo → Deploy).
